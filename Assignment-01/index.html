10 Way to be a better developer:

  1. Code almost everyday
  2. Master the fundamentals
  3. Learn on regular basis
  4. Challenge yourself
  5. Practice algorithms
  6. Engage with other developers
  7. Learn & write clean code
  8. Read other people code
  9. Contribute to open source
  10. Choose a path & stick to it
